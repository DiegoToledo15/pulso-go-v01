# Pulso GO v0.1

Demo web de Pulso GO, una propuesta de bienestar estudiantil para orientar a estudiantes según cómo se sienten durante el semestre académico.

## Tecnologías

- HTML
- CSS
- JavaScript

## Cómo ejecutar

Abrir `index.html` directamente en el navegador.

## Flujo principal

Inicio -> Check-in -> Mi Ruta -> Actividades -> Pulso del Campus -> Equipo Motor

## Nota

Esta versión no guarda datos reales. Es una demo visual y funcional para presentación universitaria.
